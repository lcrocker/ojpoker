Expand description

wiki | Poker game info

Enum representing hand “scales”, or ways in which poker hands are evaluated in different games.

Variants§

§

None = 0

None / Invalid

§

HighHand = 1

Traditional “high” poker hands

§

AceToFive = 2

Low hands, aces low, no straights or flushes

§

DeuceToSeven = 3

Low hands, aces low (“Kansas City” low)

§

AceToSix = 4

Low hands, aces high (“London” low)

§

Badugi = 5

Four cards, aces low, no matching suits

§

Badeucy = 6

Badugi, aces high

§

PaiGow = 7

High hands, except wheel beats K-high straight

§

Stripped = 8

Stripped deck: flush beats full house

§

ActionRazz = 9

Ace-to-five low, face card needed to qualify

pub const fn full_name(&self) -> fn(_: &HandValue) -> String

Game-specific function to get full English name of hand

Users will generally call these through the hand value object rather than directly. If you do call this directly, note that the function here returns a function, so you then need to call that function to get the actual name.

use onejoker::*;

let hand = Hand::new(DeckType::English).init(cards!("9s","As","9d","Ks","Ah"));
let v = ojp_high_eval_full(&hand).unwrap();
println!("{}", v.full_name());
println!("{}", HandScale::HighHand.full_name()(&v));
