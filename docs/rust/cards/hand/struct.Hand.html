    pub length: u8,

}
Expand description

wiki | Hand of cards A simple array of card objects with some utility methods. It is expected that most access will go through push()/pop(), which are fast, though things like insert() and remove() are available. Limited to 22 cards. If you need more, you can use Vec<Card>. but you lose some error checking and convenience methods.
